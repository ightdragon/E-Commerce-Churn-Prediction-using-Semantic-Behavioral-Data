# E-Commerce-Churn-Prediction-using-Semantic-Behavioral-Data
End-to-end ML pipeline predicting customer churn on the Olist e-commerce dataset (99k+ orders). Features a dual-signal churn target (180-day inactivity + fine-tuned BERT sentiment on 42k+ reviews). Random Forest champion model (0.7740 ROC-AUC) reveals delivery delays drive 66.3% of churn.
